# SPI_Demo

## 簡介
本程式範本僅用於技術應用測試，與程式碼紀錄。
### main.py
主要程式碼，可依照環境亮度改變LED顯示燈號數量，由0至3個LED燈表示。
### sensor_log.py
儲存環境數據的程式碼，未完成。
